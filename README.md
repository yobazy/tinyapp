## Dependencies
- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

#  Setup Instructions
1. Clone project into local directory
2. Navigate to directory
3. Install dependencies (npm install)
4. Run server (npm start)
5. Navigate to localhost:8080
